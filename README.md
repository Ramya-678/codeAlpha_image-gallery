 # Responsive Image Gallery

## Project Description
This is a Responsive Image Gallery project created using HTML, CSS, and JavaScript.  
The gallery displays images in a grid layout with category filter buttons such as All, Nature, and Animals.  
Users can filter images by clicking the buttons.  
The design is responsive and works on all screen sizes.

## Features
- Responsive image gallery layout  
- Category filter buttons (All, Nature, Animals)  
- Simple and clean UI  
- Works on desktop and mobile  
- Easy to customize  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Project Structure
image-gallery

## How to Run the Project
1. Download or open the project folder  
2. Open the file named image.html  
3. The image gallery will open in the browser  

## Output
The webpage shows an image gallery with filter buttons.  
Clicking the buttons displays images based on the selected category.
